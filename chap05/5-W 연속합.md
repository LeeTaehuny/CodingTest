# [5 - W] 1912번: 연속합

https://www.acmicpc.net/problem/1912

## 문제

n개의 정수로 이루어진 임의의 수열이 주어진다. 우리는 이 중 연속된 몇 개의 수를 선택해서 구할 수 있는 합 중 가장 큰 합을 구하려고 한다. 단, 수는 한 개 이상 선택해야 한다.

예를 들어서 10, -4, 3, 1, 5, 6, -35, 12, 21, -1 이라는 수열이 주어졌다고 하자. 여기서 정답은 12+21인 33이 정답이 된다.

## 입력

첫째 줄에 정수 n(1 ≤ n ≤ 100,000)이 주어지고 둘째 줄에는 n개의 정수로 이루어진 수열이 주어진다. 수는 -1,000보다 크거나 같고, 1,000보다 작거나 같은 정수이다.

## 출력

첫째 줄에 답을 출력한다.

## 예제 입력 1

```
10
10 -4 3 1 5 6 -35 12 21 -1

```

## 예제 출력 1

```
33

```

## 예제 입력 2

```
10
2 1 -4 3 4 -4 6 5 -5 1

```

## 예제 출력 2

```
14

```

## 예제 입력 3

```
5
-1 -2 -3 -4 -5

```

## 예제 출력 3

```
-1

```

## 코드

```jsx
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

// 수열의 수
int n;

// 입력받은 값 목록
int nums[100002];

// 누적합 연산용 배열
int p_sum[100002];

// 인덱스별 최소 값을 저장하기 위한 배열
int DP[100002];

/**********************************************************************************
[풀이 방법]

- DP + 누적합 문제
- 누적합으로 저장된 값들 중 최대의 범위를 구하는 문제입니다.
- 누적합에서의 최대란? : 최대 값 - 최소 값

**********************************************************************************/

int main()
{
	ios::sync_with_stdio(false);
	cin.tie(NULL);
	cout.tie(NULL);

	cin >> n;

	// 최소값을 수정하기 위한 변수
	int mn = 9999;
	// 최종 결과용 변수
	int res = -987654321;

	// 모든 값을 입력받으며 최종 결과물 초기화
	for (int i = 0; i < n; i++)
	{
		cin >> nums[i];
		res = max(res, nums[i]);
	}

	// 0번째 누적합 칸은 0번째 숫자로 설정
	p_sum[0] = nums[0];

	/* 누적합의 최대값은 최대 누적합 위치에서 나올 수 있는 최소 누적합 값을 뺀 것 */

	// 누적합 생성 및 해당 인덱스에서의 최소 누적값 저장(DP)
	for (int i = 1; i < n; i++)
	{
		// 누적합 생성
		p_sum[i] = p_sum[i - 1] + nums[i];

		// 해당 인덱스(i)에서 올 수 있는 최소 누적합 값 저장
		mn = min(mn, p_sum[i]);
		DP[i] = mn;
	}

	// 전체를 순회하며 최대 결과물 갱신
	for (int i = 0; i < n; i++)
	{
		if (res < p_sum[i] - DP[i])
		{
			if (p_sum[i] == DP[i]) continue;

			res = p_sum[i] - DP[i];
		}
	}

	// 전체를 순회하며 처음부터 자기까지 전부 더한 값과 비교하며 갱신
	for (int i = 0; i < n; i++)
	{
		if (res < p_sum[i])
		{
			res = p_sum[i];
		}
	}
	
	cout << res;
}
```
