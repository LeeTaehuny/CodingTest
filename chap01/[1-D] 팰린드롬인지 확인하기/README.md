### 팰린드롬인지 확인하기 [구현, 문자열]
---
https://www.acmicpc.net/problem/10988

[문제 풀이] 

- 문자열의 맨 앞을 가리키는 인덱스 i와 맨 뒤를 가리키는 인덱스 j를 한 칸씩 서로에게 이동시켜가며 해당 인덱스에 위치한 문자를 확인합니다.
- 만약 두 위치의 문자가 동일하다면 두 인덱스를 다음 위치로 이동시키고, 두 인덱스가 만나거나 엇갈리는 순간 true를 반환합니다.
- 만약 두 위치의 문자가 서로 다른 문자라면 while문을 종료하고 false를 리턴합니다.


[추가 사항]
- 원본 문자열을 reverse한 결과가 원본 문자열과 같은지를 비교하면 쉽습니다.
