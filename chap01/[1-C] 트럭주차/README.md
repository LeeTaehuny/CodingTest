### 트럭 주차 [구현, 시뮬레이션]
---
https://www.acmicpc.net/problem/2979

[문제 풀이] 

인덱스(분)를 저장할 101개의 공간을 가진 배열을 생성합니다.
해당 인덱스에 주차된 차량의 수를 저장할 배열을 생성합니다.
해당 시간에 주차된 차량의 수에 따라 요금을 계산해 합산합니다.


[추가 사항]
시간은 이상 미만으로 생각해야 합니다.
즉, Start와 End를 받았을 때 반복 조건은 다음과 같습니다.
	
for (int i = start; i < End; i++)
{
	cnt[i]++;
}
