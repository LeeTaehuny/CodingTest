### 곱셈 [분할 정복을 이용한 거듭제곱, 수학] (다시 풀어보기)
---
https://www.acmicpc.net/problem/1629

[문제 풀이] 

- A의 B승한 결과를 C로 나눈 나머지 값을 구하는 문제입니다.
- A, B의 값이 int 타입의 한계 값이므로, 일반적으로는 구할 수 없습니다.
- 즉, 재귀함수를 이용해 절반씩 분할해 계산하는 과정이 필요합니다.

- 여기서 중요한 점은 다음과 같습니다.
- ex) 3의 11승 % 7이라면?
- 3 % 7 * 3 % 7 * 3 % 7 * 3 % 7 * 3 % 7 * ... 3 % 7로 계산할 수 있다는 점입니다.

- 승(b)이 0승, 홀수승(a * (b - 1 / 2)), 짝수승(b / 2)을 구분해 절반씩 분할하며 결과를 계산하면 됩니다.


[추가 사항]

- 2의 4승 = 2의 2승 * 2의 2승 = n * n
- 2의 5승 = 2의 2승 * 2의 2승 * 2의 1승 = a * n * n 

- 즉, 위와 같은 방식으로 계산 양을 절반으로 줄일 수 있습니다.

- (a + b) % c = a % c + b % c
- (a * b) % c = a % c * b % c

- 즉, 최대치 20억을 20억번 곱하는 것이 아닌 각각의 나머지를 곱하고 나머지를 만드는 방식으로 저장되는 최대 값을 줄일 수 있습니다. 
- overflow 방지
