### 나는야 포켓몬 마스터 이다솜 [자료 구조, 해시를 사용한 집합과 맵] 
---
https://www.acmicpc.net/problem/1620

[문제 풀이] 

- 2개의 맵(1[이름, 숫자] / 2[숫자, 이름])을 사용해 문제를 해결했습니다.
- 입력받은 값이 문자열이면 1번 맵의 문자열에 해당하는 값을 호출합니다.
- 입력받은 값이 숫자라면 2번 맵의 숫자에 해당하는 값을 호출합니다. 

[추가 사항]

- 배열로 문제를 해결할 경우 특정 문자 or 숫자를 찾는데 드는 시간복잡도는 O(n)입니다.
- 즉, 맵으로 특정 문자 or 숫자를 찾는데 걸리는 O(log n)이 더욱 빠르므로 맵을 사용하는 것이 좋습니다.

- 입력받은 값이 문자인지 숫자인지 확인하는 방법이 중요합니다.
- atoi() 함수를 사용한 결과가 0이면 문자열이 들어온 것으로 판단합니다.
- 만약 숫자를 입력받은 경우 atoi()의 결과는 해당 숫자 그대로 출력됩니다.

string s = "1";
string s2 = "munja";

cout << atoi(s.c_str()) << '\n';
// 출력 결과 : 1

cout << atoi(s2.c_str()) << '\n';
// 출력 결과 : 0

[주의 사항]
- atoi() 함수의 결과는 입력된 숫자 or 0입니다.
- 즉, 숫자 0이 들어온다면 결과를 판별할 수가 없습니다.
