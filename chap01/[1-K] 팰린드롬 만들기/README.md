### 팰린드롬 만들기 [그리디 알고리즘, 구현, 문자열]
---
https://www.acmicpc.net/problem/1213

[문제 풀이] 

- 입력받은 문자열을 map<char, int> 컨테이너에 문자 하나씩 저장합니다.
- map에 존재하는 key(알파벳 1개)에 대응하는 value(중복숫자)가 2 이상이 되는 순간 팰린드롬은 만들 수 없습니다.

- 전부 순회하며 2 이상인 경우에는 "I'm Sorry Hansoo"를 출력하고 종료합니다.
- 홀수인 경우가 1가지라면 midStr에 해당 key를 저장합니다.
- 아닌 경우에는 leftStr 문자열에 해당 key를 더해주는 것을 반복합니다.

- 모든 순회가 끝나면 leftStr + midStr + reverse(leftStr)을 출력하는 방식으로 문제를 해결했습니다.

[추가 사항]

- 기본적인 int형 배열로 풀 수 있는 문제였습니다.
- int 타입의 arr를 만들고 해당 배열에 알파벳의 수를 저장합니다. (정렬)

- idx가 'Z'인 경우부터 역으로 'A'까지 내려가며 다음 과정을 반복합니다. (사전순)

- arr[i]가 존재한다면? (0이 아니라면) 홀수의 개수를 계산합니다.
- 홀수의 개수가 1 이상이면 종료하고, 1개면 mid에 저장합니다.
- 짝수개의 경우 idx를 2씩 증가시키며 solve의 앞과 뒤에 문자를 추가합니다.

- 모든 순회가 종료되면 mid가 있는 경우 solve의 중간 위치에 삽입하여 출력합니다.
- mid가 없는 경우에는 solve를 그대로 출력해주면 해결됩니다.
