### 농구 경기 [구현, 문자열]
---
https://www.acmicpc.net/problem/1159

[문제 풀이] 

문자열의 맨 앞 글자만을 저장하는 배열을 생성한 뒤 사전순으로 정렬합니다.
같은 글자가 배열에 있는 경우 인덱스를 저장하는 배열에 수를 추가합니다.
인덱스를 저장하는 배열에서 값이 5인 위치를 찾고, 문자열의 앞 글자를 저장하는 배열에서 해당 위치에 저장된 뮨자를 출력합니다.
boolean 변수를 하나 설정해 5인 값이 하나라도 있으면 true, 없으면 false로 지정하였고, flase인 경우 "PREDAJA"를 출력합니다.

[추가 사항]
카운트 배열을 만들어 cnt[temp[0] - 'a']에 저장하고 출력할 때 cnt가 5 이상인 것들에 대해서만 + 'a'하여 출력하면 간단하게 구현 가능합니다.
