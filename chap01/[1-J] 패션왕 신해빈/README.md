### 패션왕 신해빈 [조합론, 자료구조, 해시를 사용한 집합과 맵, 수학]
---
https://www.acmicpc.net/problem/9375

[문제 풀이] 

- 같은 부위의 옷을 중복으로 착용하는 것이 불가능합니다.
- 모든 옷을 벗고 있을 수는 없습니다.

- 위 사항을 일반화 해보면 다음과 같습니다.
- ex) 모자 2개, 바지 1개인 경우

1. 모자1을 쓰거나, 모자2를 쓰거나, 모자를 안쓰거나 : 3
2. 바지1을 입거나, 바지를 안입거나                : 2

- 3 * 2 = 6 - 1(아무것도 안 입은 경우의 수 1을 제외) = 5

- 즉, 키별로 중복되는 값을 곱하고 마지막에 1을 제외해주는 방식으로 해결하였습니다. (map 사용)

[추가 사항]

- 경우의 수는 엄청나게 커질 수도 있습니다.
- 이번 문제 풀이는 int 타입으로 받아와도 상관없었지만 앞으로는 long long타입으로 받아오는 습관을 들여야 합니다.
