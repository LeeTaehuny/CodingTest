[해결 방안]     
- 우선 호실(가로)먼저 생각해보도록 하겠습니다.
- 호실은 층 수와 관련이 있습니다.
	ex) 6층 건물에 10번째 손님이 들어온다면?
	가까운 호실을 우선적으로 배정받기에 최대 층수만큼 손님이 들어가고 다음 호실 번호로 넘어갑니다.
    즉, (n / h) + 1 = (6 / 10) + 1 = 2호실을 사용하게 됩니다.
    
- 층 수(세로)는 더욱 간단합니다.
ex) 6층 건물에 10번째 손님이 들어온다면?
가까운 호실을 우선적으로 배정받기에 층 수는 결국 손님 수를 높이로 나눈 나머지가 될 것입니다.
즉, n % h = 10 % 6 = 4층을 사용하게 됩니다.     

- 하지만 층 수(세로)와 손님의 수가 동일하다면 문제가 발생합니다.
ex) 6층 건물에 6번째 손님이 들어온다면?
간단히 생각해보면 601이라는 것을 알지만, 위 공식대로 계산하면 002가 됩니다.
즉, 층 수가 0이 되는 경우에만 층 수를 높이로 바꿔주고, 호실을 1 빼주면 해결됩니다.

- 위 공식을 사용해 문제를 해결하고 출력 시 호실은 01 ~ 99호실까지 존재하기에 2자리 수로 출력해주면 됩니다.

---
[코드]
```cpp
#include <stdio.h>

int t;

int main() {
	
	scanf("%d", &t);

	int h, w, n, y, x;

	// 테스트 케이스 만큼 입력 받기
	for (int i = 0; i < t; ++i) 
    {
    	// 높이와 넓이, 손님 입력받기
		scanf("%d %d %d", &h, &w, &n);
	
    	// 호실 구하기
		x = n / h + 1;
        // 층 수 구하기
		y = n % h;

		// 층 수와 손님의 수가 같은 경우 체크
		if (y == 0) 
        {
			y = h;
			x -= 1;
		}

		// 호실만 2자리 수로 출력
		printf("%d%02d\n", y, x);
	}
}
```
