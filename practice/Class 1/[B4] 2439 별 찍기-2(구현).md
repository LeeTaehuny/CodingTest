[해결 방안]     
- 2중 for문을 통해 구현했습니다.
- 첫 for문(i) 내부에서 2개의 for문(j, k)를 수행합니다.
- j는 (1 ~ N - i - 1)까지 순회하며 " "을 출력합니다.
- k는 (0 ~ i)까지 순회하며 *을 출력합니다.

---
[코드]
```cpp
#include <iostream>

using namespace std;

int main()
{
	int input;

	cin >> input;

	for (int i = 0; i < input; i++)
	{
        // i의 값에 따라 n-i-1개의 " "를 우선 출력합니다.
		for (int j = 1; j < input - i; j++)
		{
			cout << " ";
		}

        // i의 값에 따라 i + 1개의 *을 출력합니다.
		for (int k = 0; k <= i; k++)
		{
			cout << "*";
		}
		cout << endl;
	}
}
```
