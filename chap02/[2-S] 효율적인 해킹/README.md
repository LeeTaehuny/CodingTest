### 효율적인 해킹 [그래프이론, 그래프탐색, 너비우선탐색, 깊이우선탐색]
---
https://www.acmicpc.net/problem/1325

[문제 풀이] 
- 위 문제는 특정 노드에서 시작해서 접근할 수 있는 노드들의 수를 구하는 문제입니다.
- DFS로 접근을 하며 방문 목록을 사용해 cnt를 증가시켜 해당 노드로부터 접근 가능한 노드들의 수를 구했습니다.
- 모든 노드들에 대해 DFS를 돌리며 접근 가능한 노드 수를 배열에 저장하고, cnt들의 값들 중 가장 큰 mCnt 값인 경우에만 출력하는 방식으로 문제를 해결했습니다.
