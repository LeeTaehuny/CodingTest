### 괄호 [자료구조, 문자열, 스택]
---
https://www.acmicpc.net/problem/9012

[문제 풀이] 
- '('과 ')'으로만 이루어진 문자열을 입력받아 스택에 저장합니다.
- 비교용 스택을 만들어 스택1의 top와 스택2의 top가 쌍을 이루지 않는다면 스택2에 1의 top를 push한 후 스택1을 pop()하며, 쌍을 이룬다면 둘 다 pop() 동작을 실행합니다.
- 스택1의 내용이 빌 때까지 위 과정을 반복한 후 스택 2의 내용이 차있으면 NO, 비어있으면 YES를 출력하는 방식으로 문제를 해결했습니다.
