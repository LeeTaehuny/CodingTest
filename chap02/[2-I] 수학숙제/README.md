### 수학숙제 [문자열, 정렬, 파싱]
---
https://www.acmicpc.net/problem/2870

[문제 풀이] 

- 문자열로 입력받아 각각의 문자가 숫자이면 temp 문자열에 저장하고, 다음 문자가 숫자가 아니면 v 벡터 컨테이너에 삽입하는 방식으로 모든 숫자를 모았습니다.
- 숫자 앞에 0이 존재하는 경우를 없애기 위해 각 벡터 컨테이너 요소 별로 반복문을 돌며 숫자 앞에 0이 존재하는 경우의 수를 제거해 문제를 해결했습니다.

[오류]
- 처음에는 문자열로 입력받아 int 타입의 벡터 컨테이너에 저장하는 방식으로 풀었습니다.
- 하지만 입력 문자열의 길이가 100자 제한이므로, 99999...999 와 같이 int 자료형의 범위를 넘는 수가 들어오면 이상한 값이 출력되었습니다.
- 즉, 모든 비교와 삽입, 출력을 문자열로 진행하는 문제였습니다.
