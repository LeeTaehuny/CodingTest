### 유기농 배추 [그래프 이론, 그래프 탐색, 너비 우선 탐색, 깊이 우선 탐색]
---
https://www.acmicpc.net/problem/1012

[문제 풀이] 

- 위 문제는 연결된 컴포넌트가 몇 개 존재하는지를 요구하는 문제입니다.
- BFS, DFS를 사용해 구현이 가능하지만 저는 DFS를 사용해 구현했습니다.
- 모든 adj[][]를 반복하여 방문되지 않은 adj[][]를 DFS 알고리즘을 통해 접근하며 카운트를 증가시켜 최종 카운트를 출력해 문제를 풀었습니다.
- 인접한 위치로 지렁이가 이동 가능하므로, 방향 배열을 만들어 적용했습니다.

[오류]

- memset의 3번째 인자인 "크기" 부분에 sizeof(adj) / sizeof(int)를 전달했더니
(49, 49) 좌표가 초기화 되지 않았습니다.
- memset의 경우 sizeof(자료형) * 배열의 크기 or sizeof(배열명)으로 작성 가능하다는 점을 기억합시다..
