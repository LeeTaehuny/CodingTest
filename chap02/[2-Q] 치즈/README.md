### 치즈 [구현, 그래프이론, 그래프탐색, 시뮬레이션, 너비우선탐색]
---
https://www.acmicpc.net/problem/2636

[문제 풀이] 
- 위 문제는 0이 차지하는 영역이 1이 차지하는 영역을 DFS를 반복할 때마다 1칸씩 침범하는 느낌의 문제입니다.
- 기존 방문확인 배열 외에 bool moveAble[104][104] 배열을 만들어 만약 1의 영역에 침범한다면 해당 1의 위치를 false로 설정하는 방식으로 진행됩니다.
- 한 번의 DFS가 종료되면 치즈 영역 중 moveAble이 false인 부분을 0으로 변경하고, 남은 1의 영역의 수를 세어 배열에 저장하는 방식으로 문제를 해결했습니다.
