### 트리 [그래프이론, 그래프탐색, 트리, 깊이우선탐색]
---
https://www.acmicpc.net/problem/1068

[문제 풀이] 
- 위 문제는 인접 행렬을 사용한 DFS로 해결 가능한 문제입니다.
- 노드가 지워지는 경우 자식 노드들을 무시해야 하므로 DFS 로직 내부에서 해당 지워지는 노드는 건너 뛰고, 다른 노드들 중 자식 노드의 size가 0이거나 size가 1인데 자식 노드가 지워지는 노드인 경우에만 cnt를 증가시켜 리프 노드의 수를 구했습니다.

[오류]
- 루트 노드를 생각하지 않고 DFS를 돌려 몇 몇 경우에 에러가 발생하였습니다.
