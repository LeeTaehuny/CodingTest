### 괄호 추가하기 [구현, 브루트포스 알고리즘]
---
https://www.acmicpc.net/problem/16637

[문제 풀이] 

- 위 문제는 괄호를 추가하는 경우와 추가하지 않고 연산하는 경우를 분리해 재귀하는 완전탐색 문제입니다.

- 주요 반복 로직은 다음과 같습니다.

void go(int idx, int cal)
{
	if (idx == num.size() - 1)
	{
		res = max(res, cal);
		return;
	}

	// 하나씩 계산
	go(idx + 1, calc(idx, cal, 1));

	if (idx + 2 < num.size())
	{
		// 괄호 계산
		go(idx + 2, calc(idx, cal, 2));
	}
}

[오류]

- 처음에는 괄호를 치는 자리를 고정시켜 재귀를 시켜 몇몇 경우의 수가 무시되어 에러가 발생했습니다.
- 매 칸마다 괄호를 추가할 것인지 아니면 하나씩 연산할 것인지를 구분하는 것이 키워드였습니다.
- 괄호는 중첩되면 안되기에 괄호를 추가했다면 인덱스를 2칸씩 증가시켜 연산을 수행했습니다.
