### 컴백홈 [그래프 이론, 브루트포스 알고리즘, 그래프 탐색, 깊이 우선 탐색, 백트래킹]
---
https://www.acmicpc.net/problem/1189

[문제 풀이] 

- 위 문제는 완점탐색 알고리즘을 사용해 해결하였습니다.
- 시작 위치는 r-1, 0이며 도착 위치는 0, c-1입니다.
- 완전탐색은 r-1, 0 위치의 cnt 0부터 시작하여 이동 가능한 위치를 체크해 이동이 가능하다면 해당 위치를 방문처리하고 go(ny, nx, cnt + 1)로 재귀하며 반환시 방문해제 처리를 하였습니다.
- cnt가 k와 같고, 도착 지점이라면 ret의 수를 1 증가시키는 방식으로 문제를 해결하였습니다.
