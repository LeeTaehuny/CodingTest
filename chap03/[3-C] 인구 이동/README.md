### 인구 이동 [구현, 그래프이론, 그래프탐색, 시뮬레이션, 너비우선탐색]
---
https://www.acmicpc.net/problem/16234

[문제 풀이] 

- 위 문제는 각 영역이 인접하고, 각 영역의 인구수가 min ~ max 사이라면 합을 누적, 해당 좌표 저장, cnt를 1 증가시키는 방식으로 모든 정점을 탐색합니다. (dfs)
- 탐색이 종료된다면 탐색된 영역을 sum / cnt 값으로 변경하고 저장된 좌표와 합, cnt를 초기화합니다.
- 만약 모든 영역이 개별로 존재한다면(min ~ max 사이가 아니라면) flag는 false 상태이며, 한 칸이라도 연합이 가능하다면 true로 설정합니다.
- 1회 전체 탐색 후 flag가 true라면 연합 날짜(res)를 1 증가시키고, flag가 false라면 반복을 종료하고 최종 연합 날짜(res)를 출력해 문제를 해결했습니다.
