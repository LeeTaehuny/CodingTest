### 뮤탈리스크 [다이나믹 프로그래밍, 그래프이론, 그래프탐색, 너비우선탐색]
---
https://www.acmicpc.net/problem/12869

[문제 풀이] 

- 위 문제는 BFS 알고리즘을 사용할 수 있습니다.
- 기존 BFS의 경우 x, y 상태를 가지는 정점을 대상으로 4가지 경우의 수에 따른 반복을 진행했습니다.
- 위 문제의 경우 x, y, z 3가지의 상태를 가지는 정점을 만들고, 6가지 경우의 수에 따른 반복을 진행해 풀이가 가능합니다.
- 즉, cnt가 1일때, 2일때 ...를 계산하며 모든 scv의 체력이 0이 되는 순간이 최단거리로 구해질 수 있습니다.

- 먼저 scv의 체력을 저장할 배열을 선언합니다.
int hp[3];
- 방문 배열도 선언합니다.
int visited[64][64][64];

- 다음으로 데미지를 받을 수 있는 경우의 수를 설정합니다.
int damage[6][3] =
{
    {9, 3, 1},
    {9, 1, 3},
    {3, 1, 9},
    {3, 9, 1},
    {1, 3, 9},
    {1, 9, 3},
}

- 큐에 값을 넣어줄 때 3가지의 변수(hp1, hp2, hp3)가 필요합니다.
- 구조체를 통해 간단하게 정의해줍니다.
struct A{
    int a, b, c;
}

- BFS 로직

int bfs(int a, int b, int c) 
{
    // 제일 먼저 방문한 정점의 방문 체크
    visited[a][b][c] = 1;

    // 큐에 해당 데이터 삽입
    queue<A> q;
    q.push({a, b, c});

    // 큐 순회
    while(!q.empty())
    {
        // 큐의 맨 앞 데이터 받아오기
        a = q.front().a;
        b = q.front().b;
        c = q.front().c;

        // 큐에서 해당 정점 삭제
        q.pop();

        // 종료 조건인지 확인
        if (visited[0][0][0] != 0) break;

        // 경우에 수에 따른 방문
        for(int i = 0; i < 6, i++)
        {
            // 최소값을 0으로 고정 (max를 통해 음수를 제거)
            int nx = max(0, a - damage[i][0]);
            int ny = max(0, b - damage[i][1]);
            int nz = max(0, c - damage[i][2]);

            // 건너뛰기 조건
            if (visited[nx][ny][nz] != 0) continue;

            // 거리 누적
            visited[nx][ny][nz] = visited[a][b][c];
            // 큐에 추가
            q.push({nx, ny, nz});
        }
    }

    // 종료 조건에 의한 종료 상황
    // * 체력이 0 0 0이 되는 순간까지의 거리 - 1 반환
    return visited[0][0][0] - 1;
}

[오류]

- 저는 재귀함수를 통해 scv가 받을 수 있는 데미지 값을 반복 계산하여 문제를 풀었습니다.
- 값은 제대로 출력이 되었지만, scv의 체력을 완전탐색으로 전부 계산하며 모두 0이 될 때까지 재귀함수를 돌리다 보니 시간초과 문제가 발생했습니다.
- 다음으로 DP를 적용하여 해당 hp일 때 cnt를 저장하고 다음번에 같은 hp가 들어오게 되면 cnt보다 작은 경우에만 재귀함수를 동작하게 만들어 문제를 해결했습니다.
