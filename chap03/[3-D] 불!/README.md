### 불! [그래프이론, 그래프탐색, 너비우선탐색]
---
https://www.acmicpc.net/problem/4179

[문제 풀이] 

- 위 문제는 불이 이동하고, 사람이 이동하는 방식으로 탈출까지의 최단 거리를 구하는 문제입니다.
- bfs를 사용해 불이 각 칸에 도착하는 최소 거리를 먼저 FireV 배열에 저장합니다.
- 다음으로 사람이 각 칸에 도착하는 최소 거리를 bfs를 통해 구해줍니다.
- 사람의 최단 거리를 구할 때 불과 겹치지 않도록 FireV[ny][nx] > visited[y][x]인 경우에만 큐에 삽입해주는 방식으로 탈출 시간을 구해 문제를 해결했습니다.
