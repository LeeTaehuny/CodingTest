### 주난의 난 [그래프 이론, 그래프 탐색, 데이크스트라, 0-1 너비 우선 탐색]
---
https://www.acmicpc.net/problem/14497

[문제 풀이] 

- 위 문제는 이동할 수 없는 구역을 만나면 체크해놓고 다음 차례에 이동이 가능하도록 수정해 목적지까지 도달할 때까지 반복해서 DFS 알고리즘을 수행하는 문제입니다.
- 1. 기존 dfs 알고리즘에서 벽(1)을 만나면 체크(벡터 컨테이너에 해당 벽의 좌표 저장)합니다.
- 2. dfs문이 종료되었다면 체크된 영역을 전부 이동 가능한 길(0)으로 변경합니다.
- 3. 목적지(#)를 찾을 때까지 반복하며 카운트를 증가시키고 목적지에 도달했다면 반복문을 종료합니다.
