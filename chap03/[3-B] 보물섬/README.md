### 보물섬 [그래프 이론, 브루트포스 알고리즘, 그래프 탐색, 너비 우선 탐색]
---
https://www.acmicpc.net/problem/2589

[문제 풀이] 

- 위 문제는 가장 멀리 떨어진 육지의 거리를 구하는 문제입니다.
- 모든 육지와 바다를 입력받으며 해당 입력값이 육지인 경우 배열에 위치를 저장합니다.
- 육지들을 저장한 배열을 하나씩 순회하며 해당 육지로부터 BFS 탐색을 시작합니다.
- 해당 육지에서 탐색 가능한 마지막 육지까지의 거리를 변수에 저장하며, 기본 거리(-1)와 비교해 더 큰 값을 저장합니다.
- 모든 육지를 대상으로 순회를 하며 가장 먼 거리를 출력하는 방식으로 문제를 해결했습니다.
