### 현욱은 괄호왕이야!! [자료구조, 스택]
---
https://www.acmicpc.net/problem/15926

[문제 풀이] 

- 위 문제는 문자열에서 성공적으로 완성된 괄호의 연속된 최대 길이를 구하는 문제입니다.
- 즉, ())()()과 같다면 답은 ()()인 4가 나오게 됩니다.
- 문자열을 폭파 시킬 때는 대부분 스택 자료구조를 사용하면 편합니다.
- 또한, 문자열을 입력받으면서 문자열의 길이와 같은 boolean 배열을 생성하여, 해당 괄호가 성공적으로 폭파된다면 해당 괄호가 위치한 인덱스의 boolean 배열의 값을 true로 설정합니다.
- 모든 괄호를 폭파 시켰다면, boolean 배열을 순회하며 연속된 true를 찾아주면 문제가 해결됩니다.
